Medicus Dictate

Local voice dictation for clinical work on Windows. Press a hotkey, talk, and your words appear wherever your cursor is.

Patient data never leaves your PC

What it does

Medicus Dictate sits quietly in your system tray. Press Ctrl + Alt + Space, speak a consultation note, press again. The transcribed text is dropped at your cursor — whether that's in EMIS, SystmOne, Vision, Outlook, a browser, or any other text field.

Transcription runs locally on your PC using Whisper. There is no account to create, no cloud service, no subscription. It is designed for clinicians who can't send patient-identifiable text to third-party services.

Features

Tap or hold

A quick tap toggles recording. Hold the same hotkey for push-to-talk so the mic isn't hot while you speak to the patient.

Live mic feedback

The tray icon pulses with your voice level and shows a peak bar. You'll know immediately if the mic is muted or the wrong one.

Voice commands

Say "new line", "full stop", "caps on … caps off", "scratch that" to undo, or "read that back" to hear it.

Clinical vocabulary

Drug names, colleague names, local jargon — add them in a config file and Whisper is biased toward them. Per-app profiles too.

Medical post-processing

"Fifty micrograms once daily" becomes "50 mcg OD". Optional BNF shorthand, custom dictionaries, smart grammar tidy.

No cloud, ever

Everything runs on your machine. Optional local LLM tidy via Ollama — the app refuses any non-private endpoint.

Privacy by design

No audio, no text, no telemetry leaves your computer. The Whisper model is downloaded once from HuggingFace on first run; after that, the app works entirely offline.

  • No account, no login, no API key.
  • No analytics or crash reporting.
  • Clipboard contents are saved and restored around each paste.
  • Optional smart-tidy mode uses a local Ollama endpoint only — non-private IPs are refused.
  • Source code is open and auditable; no obfuscated binaries.

Getting it running

Three steps. You do need Python 3.10+ installed first — make sure you tick Add Python to PATH in the installer. Everything after that is double-clicks; no terminal required.

  1. Download and unzip

    Click the Download (ZIP) button, then right-click the downloaded file → Extract All… to somewhere you'll remember (Documents is fine).

  2. Double-click setup.bat

    This runs once, on first install. It creates a self-contained Python environment inside the folder and installs the dependencies. Takes a couple of minutes. You'll see a black window with progress — that's expected.

  3. Double-click run.bat

    This starts Medicus Dictate. The first run downloads the speech model (about 470 MB) — one-off. After that, launches are instant.

    Look for a small grey circle in your system tray (bottom-right of the screen, near the clock). You can pin it for easy access.

  4. Dictate

    Press Ctrl + Alt + Space to start recording, press again to stop. The transcription appears wherever your cursor is — in EMIS, an email, anywhere.

Want it to launch automatically at sign-in? Drag a shortcut to run.bat into the Startup folder (Win + R, type shell:startup, Enter).

Locked-down Windows (NHS, corporate)

Many NHS trusts and corporate images block unsigned .exe installers outright — Windows Defender, SmartScreen, or AppLocker will refuse to run them and you often can't override that without admin rights. This tool is deliberately designed around that constraint. There are no binaries to be blocked; the app is a set of Python scripts launched by batch files, both of which are plain text.

If your trust doesn't let you install Python, you don't need admin rights to use a portable Python. Download WinPython (64-bit), extract the ZIP anywhere you have write access (Documents, Desktop), rename the inner python-3.x.x.amd64 folder to just python, and drop it next to setup.bat. setup.bat detects it automatically and uses it.

Whisper's speech model is downloaded from Hugging Face on first launch. If your trust proxies or blocks Hugging Face, download the model on a home PC and copy the ~\.cache\huggingface folder across — the app will pick it up without any further network access.

Common questions

Is my patient data sent anywhere?

No. Transcription runs on your PC. The only network activity is a one-off download of the Whisper model on first launch.

Does it work with EMIS, SystmOne, Vision?

Yes. The app pastes into whatever window has keyboard focus, so any text field works. Per-app profiles let you tune the vocabulary and formatting rules per clinical system.

What hardware do I need?

Windows 10 or 11, a microphone, and roughly 2 GB of free disk space for the Whisper model. An NVIDIA GPU is optional and speeds up transcription, but the default configuration uses the CPU.

Will Windows Defender flag it?

If you build a .exe with PyInstaller yourself, Defender may flag unsigned binaries. Running the app directly with python -m src avoids this entirely.

Can I change the hotkey?

Yes — edit config.toml next to the app. Any combination of modifiers plus a key, in pynput's syntax.

Why is there no period, colon, or backslash command?

"Period" collides with the menstrual / temporal sense in clinical dictation; "colon" collides with the anatomy. Use "full stop" for a period and type a colon literally. Keeping the command list conservative reduces false triggers in real consultation notes.
